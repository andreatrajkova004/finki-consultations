## 📋 Опис

Апликацијата овозможува професорите да креираат термини за консултации, а студентите да ги резервираат со избор на сопственото времетраење. Повеќе студенти можат да резервираат во ист термин — секој зазема само онолку минути колку му се потребни.

---

## ✨ Функционалности

**Професор:**
- Креира слотови за консултации (датум, час, времетраење, локација, напомена)
- Гледа кој студент резервирал и во кое точно време
- Брише слотови
- Десен панел со листа на сите резервации

**Студент:**
- Гледа сите достапни термини во неделен календарски приказ
- Филтрира термини по професор
- Резервира со избор на времетраење (брзи копчиња: 10, 15, 20, 30, 45, 60 мин)
- Откажува сопствена резервација
- Десен панел со листа на сопствените резервации

## 🔧 Технологии

- **React 18** — frontend интерфејс, hooks, компонентна архитектура
- **Firebase Authentication** — автентикација со email/password
- **Cloud Firestore** — NoSQL база во реално време
- **date-fns 3** — работа со датуми, македонска локализација
- **React Router 6** — навигација меѓу страниците

## 🗃️ Firestore структура

Сите податоци се чуваат во колекцијата `slots`. Резервациите се чуваат како array внатре во слот документот:

```json
{
  "professorId": "uid",
  "professorName": "Проф. Петровски",
  "startTime": "2026-03-10T09:00:00",
  "endTime": "2026-03-10T11:00:00",
  "duration": 120,
  "location": "Кабинет 215",
  "notes": "Носете го проектот",
  "bookings": [
    {
      "studentId": "uid",
      "studentName": "Ана Бањанска",
      "startOffset": 0,
      "durationMins": 20,
      "bookedAt": "2026-03-09T10:00:00"
    }
  ]
}
```

## 🔥 Firebase Setup

### Чекор 1 — Вклучи Authentication
Firebase Console → **Authentication** → **Get started** → **Email/Password** → Enable

### Чекор 2 — Создај Firestore Database
Firebase Console → **Firestore Database** → **Create database** → Production mode

### Чекор 3 — Постави Firestore Rules
Firebase Console → **Firestore** → **Rules** → залепи ја содржината од `firestore.rules`

### Чекор 4 — Додај Firebase config
Залепи го твојот config во `src/firebase/config.js`:

```js
const firebaseConfig = {
  apiKey: "...",
  authDomain: "...",
  projectId: "...",
  storageBucket: "...",
  messagingSenderId: "...",
  appId: "..."
};
```
